# 시각 장애인 사용자를 위한 음성 명령 예시 - 네이버 이용 시나리오

## 실제 사용자가 사용할 법한 음성 명령어 10가지

### 1. 날씨 관련
명령어: "오늘 날씨는 어때?"
예상 경로: 네이버 메인 → 날씨 → 현재 날씨
설명: 가장 많이 사용되는 일상적인 질문

### 2. 웹툰/만화 추천
명령어: "만화 추천해줘"
예상 경로: 네이버 메인 → 웹툰 → 오늘의 웹툰 → 로맨스/액션 장르
설명: 여가시간 콘텐츠 탐색

### 3. 최신 뉴스 확인
명령어: "최근에 무슨 일이 있었어?"
예상 경로: 네이버 메인 → 뉴스 → 사회 뉴스
설명: 시사 정보 습득

### 4. 지역 맛집 검색
명령어: "강남역 근처 맛집 찾아줘"
예상 경로: 네이버 메인 → 검색창 → 지역 검색 → 지도
설명: 외출 준비 및 식당 정보 탐색

### 5. 생활용품 구매
명령어: "샴푸 어디서 싸게 살 수 있어?"
예상 경로: 네이버 메인 → 쇼핑 → 생활용품 → 화장품/미용
설명: 일상 필수품 구매

### 6. 교통 정보 확인
명령어: "집에서 병원까지 어떻게 가?"
예상 경로: 네이버 메인 → 지도 → 길찾기
설명: 이동 경로 및 교통편 확인

### 7. 건강 정보 검색
명령어: "감기에 좋은 음식이 뭐야?"
예상 경로: 네이버 메인 → 검색창 → 블로그 검색
설명: 건강 관련 정보 탐색

### 8. 스포츠 결과 확인
명령어: "어제 야구 경기 결과 알려줘"
예상 경로: 네이버 메인 → 뉴스 → 스포츠 뉴스
설명: 스포츠 관심사 확인

### 9. 요리 레시피 찾기
명령어: "간단한 저녁 요리 알려줘"
예상 경로: 네이버 메인 → 검색창 → 블로그 검색 → 요리 블로그
설명: 식사 준비 정보

### 10. 커뮤니티 소통
명령어: "비슷한 관심사 사람들과 이야기하고 싶어"
예상 경로: 네이버 메인 → 카페 → 관심 주제 카페
설명: 사회적 소통 및 정보 공유

## 각 시나리오별 접근성 고려사항

### 우선순위가 높은 기능들 (접근성 점수 90+ 기준)
1. 검색 기능 (98점) - 키보드 단축키 Ctrl+L 지원
2. 메인 페이지 (95점) - 명확한 탭 네비게이션
3. 날씨 정보 (92점) - 빠른 로딩과 음성 읽기 최적화

### 중간 우선순위 기능들 (80-89점)
1. 뉴스 (85점) - 헤드라인 스크린리더 최적화
2. 쇼핑 (80점) - 상품 정보 구조화

### 개선 필요 기능들 (80점 미만)
1. 웹툰 (75점) - 이미지 중심 콘텐츠로 대체 텍스트 필요
2. 지도 (70점) - 시각적 정보를 음성으로 변환하는 기능 부족

## 테스트 시나리오 예시

### 시나리오 1: 날씨 확인 → 외출 준비
1. 사용자: "오늘 날씨는 어때?"
2. 시스템: 네이버 메인(1초) → 날씨 섹션(1초) → 현재 날씨(1초)
3. 음성 피드백: "현재 서울 기온 15도, 맑음입니다"
4. 후속 액션: "우산 챙겨야 하나?" → 주간 예보로 자동 이동

### 시나리오 2: 웹툰 추천 → 독서
1. 사용자: "만화 추천해줘"
2. 시스템: 네이버 메인 → 웹툰 → 오늘의 웹툰
3. 음성 피드백: "오늘 업데이트된 웹툰 5개를 찾았습니다"
4. 후속 액션: 장르별 필터링 또는 인기 웹툰 추천

### 효과성 측정 지표
- 평균 응답 시간: 3초 이내
- 정확도: 목표 페이지 도달률 85% 이상
- 사용자 만족도: 음성 피드백 명료성
- 접근성 점수: 키보드만으로 전체 기능 이용 가능
